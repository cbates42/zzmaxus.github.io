Inari Level

I blocked out a small level for Inari, a game I'm working on at Columbia. The main idea that was decided for the first level is that it would take place at a festival, and have the player defeat the demons around the town. At first I kind of wanted to have some non-linearity in the level, but I decided to just have it be a shorter linear level. I got through it kind of quickly, but I think a newer player would take a little longer to get used to how the combat and movement is. Especially the walljump. One thing I would want to do is add a longer rooftop section after the current end of the level. You can see me walk through the level here: https://youtu.be/omaQK3bKmjY And you can find Inari here: https://bentocat.itch.io/inari

Animation from a spritesheet

I recently tried out learning more about using spritesheets to create 2D animations in Unity, and managed to use one pre-made spritesheet to do so. Using this Nightborne Warrior by CreativeKind spritesheet: https://creativekind.itch.io/nightborne-warrior I was able to create a few animations to be used in unity. Using these sprites I created the animations seen in this video: https://www.youtube.com/watch?v=n7vyjnQDC7E I mainly chose this spritesheet because I thought it'd be interesting to use as inspiration for another idea I have, and would likely give a decent base for that idea.
